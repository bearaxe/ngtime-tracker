Features


Undo Delete: Accidentally delete something? Refresh the page before touching a button and it'll be back from the graveyard. This doesn't work if you push save or update the project list, so be careful.
AutoPause: Click any project's start button to pause any already running timer
Pin Project: Pin a project to prevent it from being deleted if you hit the Delete All button
Quick List: Click a project in the quick list to start its timer
CSV: You can download a csv copy of the project list in the Options menu
